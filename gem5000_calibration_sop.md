# GEM 5000 Calibration SOP

## Purpose
This procedure outlines the basic steps for calibrating a GEM 5000 landfill gas analyzer to support accurate field readings.

## Equipment and Materials
- GEM 5000 analyzer
- Calibration gas
- Flow regulator
- Tubing
- Fresh air source

## Procedure

### 1. Start Up the Analyzer
1. Press and hold the power button.
2. Allow the analyzer to complete startup and self-check.
3. Disconnect the charger before use.

### 2. Log In
1. Select the appropriate username.
2. Avoid using another operator’s login.

### 3. Access Gas Check Menu
1. Navigate to Operation Settings.
2. Select Gas Check.

### 4. Fresh Air Check
1. Disconnect all tubing.
2. Select Fresh Air.
3. Allow the pump to run.
4. Select Check.
5. Select Calibrate once prompted.

### 5. CH4 / CO2 Calibration
1. Select the correct calibration gas mixture.
2. Connect the regulator and tubing.
3. Allow readings to stabilize.
4. Select Check.
5. Select Calibrate after completion.

### 6. O2 Calibration
1. Select the correct oxygen calibration gas.
2. Connect the regulator and tubing.
3. Allow readings to stabilize.
4. Select Check.
5. Select Calibrate.

## Notes
Calibration should be completed before field use. This document is for educational and portfolio purposes only and does not replace manufacturer instructions or employer-approved procedures.
